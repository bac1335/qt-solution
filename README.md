# qt-solution
qt-solutions提供了8个开源项目其实这是官方提供的源代码
----------------------------------------------------------------------
这8个项目分别是： 
qtbrowserplugin  浏览器插件
qtlockedfile         锁定文件
qtpropertybrowser 属性编辑器
qtscriptclassic       脚本支持
qtservice              提供服务属性
qtsingleapplication 单例运行
qtsoap                  SOAP协议
qtwinmigrate        混合运行，主要是指MFC吧。
